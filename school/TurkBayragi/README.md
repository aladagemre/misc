Bu program, Java ile Türk bayrağı çizer. Bir okul projesi olarak hazırlanmıştır. Bayrağa tıklandığında konsolda tıklandığına dair bir mesaj gelmektedir.

Çalıştırmak için:

javac TurkBayragi.java
java TurkBayragi
